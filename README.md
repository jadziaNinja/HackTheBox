# HackTheBox
Tracking the box

This repository documents my hands-on cybersecurity learning through capture the flag competitions, lab environments, and security challenges. Each writeup covers a completed room, machine, or challenge with notes on methodology, tools used, and lessons learned along the way.
I work through challenges on platforms including Hack The Box, TryHackMe, and SANS competitions including BootUp CTF and NetWars SkillsQuest. My background is in VFX pipeline engineering, and I transitioned into cybersecurity through the SANS WiCyS scholarship program, earning my GCIH, GSEC, and GFACT certifications. These writeups are part of how I build and document that practice.
The goal is not just to solve the challenge but to understand why the solution works, what the underlying concept is, and how it connects to real-world security operations. If something in here is useful to you, that is a bonus.
